# 台灣正體中文 AI 詞彙與句式警示列表

以下詞彙在 AI 生成的中文文本中出現頻率異常偏高。遇到時必須替換或刪除。

## AI 高頻詞替代表

| AI 高頻詞 | 替代 |
|-----------|---------|
| 此外 | 也、另外、還有（或直接刪除） |
| 至關重要 | 重要、很重要（或具體說明為何重要） |
| 深入探討 | 討論、分析、看看 |
| 強調/突出/彰顯 | 提到、指出、說（或直接刪除） |
| 持久的/永恆的/持續的 | 一直、長期、到現在還 |
| 增強/提升 | 改善、變好、讓……更好 |
| 培養/促進/涵養 | 練、學、養成 |
| 獲得/贏得 | 拿到、得到 |
| 格局（抽象名詞） | 情況、狀況、現況 |
| 關鍵性的/樞紐性的 | 重要的（或具體說明） |
| 展示/展現 | 讓人看到、呈現 |
| 織錦/畫卷（比喻） | 直接刪除，用具體描述取代 |
| 寶貴的 | 有用的、有價值的 |
| 充滿活力的 | 熱鬧的、活潑的（或具體描述） |
| 不可磨滅的 | 深遠的、很大的 |
| 相互作用/交織 | 互相影響、一起 |
| 複雜性/錯綜複雜 | 複雜、麻煩 |
| 無縫的/無縫銜接 | 順暢的、順利的 |
| 賦能/賦予力量 | 幫助、讓……能夠 |
| 奠定基礎 | 打底、開始 |

## AI 句式模板

遇到以下句式必須改寫或刪除：

| AI 句式 | 處理方式 |
|---------|----------|
| 「在當今……的時代」 | 刪除，直接進入主題 |
| 「隨著……的快速發展」 | 簡化或刪除 |
| 「值得一提的是」 | 直接說 |
| 「不容忽視的是」 | 直接說 |
| 「眾所周知」 | 刪除，直接陳述 |
| 「總而言之/綜上所述」 | 簡化或刪除 |
| 「……的重要性不言而喻」 | 具體說明為什麼重要 |
| 「這不僅……更是……」 | 違反鐵律。直接陳述。 |
| 「從……到……，從……到……」 | 舉一兩個具體例子 |
| 「讓我們一起……」 | 刪除或直接說明行動 |
